## Session 4 ‚Äî 2026-04-03 ¬∑ build_bootstrap.py ‚Äî factory discovery + changelog injection

### Added

- `FactoryDiscoveryError` ‚Äî custom exception class carrying `unmatched`, `candidates`,
  and `valid` path lists; lets the caller recover with the files that passed while
  surfacing actionable rename hints for those that didn't
- `discover_factories(scan_dir)` ‚Äî recursive `.md` scanner enforcing the `<dir>/<dir>.md`
  naming convention; silently ignores `README.md` (any casing) and anything under
  `versions/` at any depth relative to the scan root; raises `FactoryDiscoveryError`
  for files that don't match, including both the offending paths and expected filenames
- `parse_factory_entry(path, source_label)` ‚Äî extracts a registry-compatible dict from
  a factory `.md` file; prefers the `## Metadata` JSON block (present in all
  well-formed factories); falls back to first heading + filename with `autogenerated: true`
  for bare/draft files; always stamps `source` and `path` for LLM provenance
- `--include-impromptu` flag ‚Äî opt-in; discovers factory `.md` files from
  `prompts/impromptu/factories/` and appends them to PASTE-1 as a clearly-labelled
  `IMPROMPTU FACTORIES` block (separate from registry-verified entries)
- `--impromptu-dir` argument ‚Äî override the impromptu factories directory relative to
  `--root` (default: `prompts/impromptu/factories`)
- `--no-changelog` flag ‚Äî suppress CHANGELOG.md injection into PASTE-2; changelog is
  included automatically when found at the repo root
- CHANGELOG.md auto-discovery ‚Äî the script now searches for `CHANGELOG.md` at the
  repo root; when found (and not suppressed), its content is appended to PASTE-2 so
  the LLM session has full project history as context

### Changed

- `render_paste1(factory_rows, impromptu_rows=None)` ‚Äî now accepts optional
  `impromptu_rows`; appends a separate labelled block when provided so discovered
  and registry-verified factories are never silently mixed
- `render_paste2(profile, orchestrator, changelog=None)` ‚Äî now accepts optional
  `changelog`; appends changelog content under its own `--- CHANGELOG.md ---` fence
  when provided
- `validate_paths` ‚Äî CHANGELOG.md added to checklist as an optional entry (prints
  `OPTIONAL` rather than `MISSING`; does not fail validation); impromptu dir check
  added when `--include-impromptu` is active
- `main()` ‚Äî wraps `discover_factories` in `try/except FactoryDiscoveryError`; prints
  warning with rename hints and continues with valid files; reads and reports
  CHANGELOG.md when present and not suppressed via `--no-changelog`
- Discovery convention hardened: factory `.md` stem must match its immediate parent
  directory name ‚Äî convention-based, no hardcoded filename lists except `readme.md`
  and the `versions/` directory rule

## Session 3 ‚Äî 2026-04-02 ¬∑ build_bootstrap.py refactor

Code-quality pass on `build_bootstrap.py`. No behavior change.

### Changed

- `build_bootstrap.py`
  - Removed `read_text()` one-liner wrapper; replaced with inline `.read_text(encoding="utf-8").strip()` at the two call sites in `main()`
  - Removed `write_bootstrap_quickstart()`; content promoted to a module-level `QUICKSTART` constant and merged into the `outputs` dict ‚Äî one less function, one fewer file write call
  - Removed `resolve_paths()`; path construction inlined directly in `main()` ‚Äî the dict abstraction added indirection without reuse benefit
  - Merged `resolve_paths` + `validate_paths` into a single `validate_paths(root, reg)` ‚Äî check list is a flat `list[tuple[Path, str]]`, no separate labels dict
  - Fixed bug: `factory_rows` / `strategy_rows` were read twice when `--validate` was passed (once for the count printout, once unconditionally before the output block); now read once before the `args.validate` branch
  - Renderer functions: switched from `dedent(f"""...""")` to plain f-string concatenation ‚Äî `dedent` only helps when literals are indented inside a function body; these are not
  - `render_paste3`: collapsed council/core sort into a single ternary append ‚Äî `(council if "council" in s.get("tags", []) else core).append(line)`
